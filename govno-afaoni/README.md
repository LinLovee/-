# govno-afaoni

Телеграм-бот с текстовой RPG-игрой по мотивам вселенной «Токийский гуль».

## Что добавлено по механикам
- прокачка за **RC-клетки** (`/evolve`), RC добываются в PvP и PvE;
- RC за убийство другого игрока (`/attack`);
- RC за убийство/пожирание NPC-человека (`/eat`) с риском засады CCG;
- **гача** (`/gacha`) с редкостями `COMMON/RARE/EPIC/LEGENDARY`;
- рейды на район (`/raid`) как дополнительный цикл прогрессии;
- выбор кагуне на старте с подробным описанием.

## Все типы кагуне (выбор на старте)
Создай персонажа командой `/start <тип>`:

- `rinkaku` (Ринкакку): щупальца, агрессия и регенерация.
- `ukaku` (Укаку): скорость и мобильность.
- `koukaku` (Коукаку / Кукаку): броня и выживаемость.
- `bikaku` (Бикаку): сбалансированный универсал.

Если ввести просто `/start`, бот покажет подробное описание каждого типа и пример выбора.

## Команды
- `/profile` — профиль
- `/hunt` — охота (кд 24ч)
- `/eat` — пожирание человека (кд 12ч)
- `/raid` — рейд на район (кд 6ч)
- `/train` — тренировка за йены
- `/evolve <strength|stamina|hp>` — мутация за RC
- `/gacha` — гача-крутка
- `/attack <id>` — PvP атака
- `/top` — рейтинг игроков

## Локальный запуск
1. Создайте бота в BotFather и получите токен.
2. Скопируйте `.env.example` в `.env` и вставьте токен.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите:
   ```bash
   python bot.py
   ```

## Деплой на Render.com
Рекомендуется хостинг как **Background Worker**.

1. Создайте `Background Worker` в Render.
2. Build Command: `pip install -r requirements.txt`
3. Start Command: `python bot.py`
4. Установите env-переменные:
   - `BOT_TOKEN`
   - `DB_PATH=/var/data/game.db`
5. Подключите **Persistent Disk** для сохранения SQLite между рестартами.

## Переменные окружения
- `BOT_TOKEN` — токен Telegram-бота.
- `DB_PATH` — путь к SQLite базе (по умолчанию `game.db`).
